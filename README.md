# 🚀 Navy Notification Agent

An automated Python-based agent that monitors the Indian Navy official website for recruitment updates and sends email alerts.

## 🔥 Features

- Scrapes latest news from Indian Navy website
- Extracts only relevant news section
- Sends email alerts automatically
- Cloud deployed (runs 24/7)
- Can be scheduled weekly

## 🛠️ Tech Stack

- Python
- BeautifulSoup (Web Scraping)
- Requests
- SMTP (Email Automation)
- Cloud Deployment (Render)

## 📂 Project Structure

navy_agent/
│── main.py
│── scraper.py
│── notifier.py
│── config.py
│── requirements.txt

## ⚙️ How it Works

1. Fetches data from Indian Navy website
2. Extracts news section
3. Processes text
4. Sends email notification
